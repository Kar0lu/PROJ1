## Creating a Virtual Environment
### Windows
```
python -m venv myenv
myenv\Scripts\activate
pip install -r requirements.txt
```
### Linux/ macOS
```
python -m venv myenv
source myenv/bin/activate
pip install -r requirements.txt
```
